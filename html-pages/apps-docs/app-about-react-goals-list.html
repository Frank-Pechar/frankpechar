Goals List App - React Project

App Description

App Overview

This App Provides for Adding and Deleting Goals to a List

User Functionality

  • Add a Goal to List
  • • Input goal item
  • • Click <Add Goal> Button
  • Delete a Goal from list
  • • Click on the goal item in the list

Coding Features and Methods Used

This App was an Exercise into Various Styling Methods Such As: • Styled Components • Styled Components using dynamic props • CSS Modules • CSS Modules using dynamic styles • Dynamic Inline CSS styles • Toggling CSS classes
