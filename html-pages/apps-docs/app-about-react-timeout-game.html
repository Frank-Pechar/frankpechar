Timeout Game - React Project

App Description

App Overview

Timeout Game

User Functionality

  • Add Player Name
  • • Enter Name in Set Name Input Field
  • • Click <Set Name> Button
  • Play a Game
  • • Click <Start Challenge> Button Next to Game Time
  • • Guess by Waiting a Specific Amount of Time
  • • Click <Stop Challenge> Button
  • • A Score will be Displayed
  • • Click <Close> Button

Coding Features and Methods Used

This App Uses React Features Such As: • Usage of Refs as element pointers • Usage of Refs as persistent instance variables • Forwarding Refs to other components • Implemented useImperativeHandle to allow access to child (ResultModal) function within parent (TimerChallenge) to open modal • Use of Portal to render modal into DOM • Implemented useState hook • Use of Set State callback function parameter to ensure most current state • Passing props • Maintaining minimum state management • Dynamic styling • Basic event handling
