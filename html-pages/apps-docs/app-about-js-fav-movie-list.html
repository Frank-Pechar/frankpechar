Favs Movie List App - Javascript Project

App Description

App Overview

This App Provides for Creating a List of Favorite Movies

User Functionality

  • Add Favorite Movies to List
  • • Click the <ADD MOVIE> Button
  • • Add Movie Title, Image URL, Rating from 1 to 5
  • • Click the <ADD> Button
  • To Delete Movie from List
  • • Click on the Movie Item in the List
  • • On the Confirm Delete Modal - Click the <Yes> Button

Coding Features and Methods Used

• Basic DOM manipulation and data entry using modals and class manipulation
