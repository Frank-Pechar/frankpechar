App Description

App Overview

Tic-Tac-Toe Game

User Functionality

  • Change Player Name
  • • Click <Edit> Button Next to Player Name
  • • Enter: Name
  • • Click < Save> Button
  • Enter a Turn Selection
  • • Click a Blank Box Inside Game Box
  • Start a New Game
  • • Click <Rematch> Button

Coding Features and Methods Used

This App Uses React Features Such As: • Implemented useState hook • Use of state update function parameter within Set State Function to ensure most current state • Two way binding of input fields • Lifting State Up to parent component • Use of list Keys for unique id • Minimum State Management • Deriving and Lifting Computed Values • Dynamic styling • Usage of props • Multi-Dimensional arrays and lists
