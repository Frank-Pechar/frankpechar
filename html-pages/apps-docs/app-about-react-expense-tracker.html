Expense Tracking Log App - React Project

App Description

App Overview

This App logs and tracks expenses and summarizes them by month and year • New expenses can be added • An itemized list and a monthly chart for expenses can be viewed

User Functionality

  • List Monthly Expenses for a Given Year
  • • Select a year from the - Filter by year - drop down menu
  • Add New Expense to Log
  • • Click <Add New Expense> Button
  • • Enter: Title, Amount, Date
  • • Click <Add Expense> Button

Coding Features and Methods Used

This App Uses React Features Such As: • Passing props data by object or multiple values to components • Use of props.children • Implemented useState hook for updating data • Managed multiple states with one object state or individual state fields • Use of state update function parameter within Set State Function to ensure most current state • Two way binding of input fields • Lifting State Up to parent component • Used Derived value based on related State fields • Use of list Keys for unique id • Styling wrapper component • Dynamic styling • Basic event handling
