Project Planner App - Javascript Project

App Description

App Overview

This App Provides Drag and Drop Functionality and Positioning of Screen Elements

User Functionality

  • Move Projects Between Active and Finished Lists
  • • Drag and Drop Projects between Lists
  • • Can also Click <Finish> or <Activate> Button to Move Items
  • Display Tooltip Popup
  • • Click the <More Info> Button to Display a Tooltip Popup
  • • Click the Tooltip Popup to Remove it

Coding Features and Methods Used

• This Coding Project was a Study of Drag and Drop and Element Positioning • This App Uses Javascript Features Such As: Base and Sub Classes for Created Objects • Moving HTML Elements within the DOM • Element Positioning • Static Methods • Originally Designed Just for Desktop - Made Necessary Adaptations for Mobile Devices
