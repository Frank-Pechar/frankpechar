User Name and Age List App - React Project

App Description

App Overview

This App Provides for Creating a User Name and Age List

User Functionality

  • Add a Username and Age to List
  • • Enter User Name and Age
  • • Click <Add User> Button

Coding Features and Methods Used

This App Uses React Features Such As: • Passing props data • Use of props.children • Input data error handling and error modal popup • Implemented useState hook for updating data • Use of state update function parameter within Set State Function to ensure most current state • Two way binding of input fields • Lifting State Up to parent component • Use of list Keys for unique id • Styling wrapper component • Basic event handling
